# Product Overview

Samsung Display Shop is an e-commerce application for purchasing Samsung LCD and AMOLED displays. The platform provides a complete shopping experience with product browsing, cart management, user authentication, order processing, and administrative capabilities.

## Key Features

- Product catalog with filtering and search
- Shopping cart and favorites functionality
- User authentication and profiles
- Order management and tracking
- Admin panel for product and order management
- Excel import/export capabilities for products
- Responsive design with dark/light theme support

## User Roles

- **Customers**: Browse products, manage cart/favorites, place orders, track shipments
- **Administrators**: Manage products, process orders, import/export data, view analytics

The application focuses on providing a streamlined experience for purchasing display components with robust administrative tools for inventory management.
